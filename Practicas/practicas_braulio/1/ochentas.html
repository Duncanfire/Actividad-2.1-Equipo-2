== ERA DE NEGOCIOS ==


1980. En este año es posible comprar ordenadores de uso personal. Los computadores de uso personal se usan para hojas de cálculo y procesadores de texto.


En el mismo año, se crea el lenguaje de programación HTML


1981. Se lanza el IBM PC para el ecosistema de internet.


1982. Se desarrolla el protocolo de internet TCP/IP.


1983. Sale una versión mejorada del lenguaje de programación C. Dicho lenguaje mejorado es C++.


1986. Nace el proyecto por parte de Michael Stonebraker de la base de datos: PostgreSQL.


1987. Larry Wall diseña el lenguaje de programación PERL.


1989. Tim Berners-Lee, concibe la World Wide Web.
